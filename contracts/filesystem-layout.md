# Filesystem Layout Contract

Status: placeholder

Defines shared vocabulary for user-facing storage locations without assuming POSIX.

Candidate logical roots:

- `home`
- `apps`
- `config`
- `temp`
- `assets`
- `logs`

Project examples:

- RetroTUI: host filesystem paths.
- RetroDesk: host filesystem paths.
- WiiOS: SD/USB roots.
- ArmoniOS: bootfs/tmpfs/FAT32 paths.
